## Etapas del Proyecto

### Etapa 1: Planeación y Recopilación
- Selección de dataset (12+ atributos, 2000+ registros)
- Identificación de fuente de datos
- Definición del contexto del problema
- Formulación de pregunta guía analítica

### Etapa 2: Exploración
- Carga de datos
- Análisis exploratorio
- Diagnóstico de calidad de datos
- Visualizaciones y correlaciones

### Etapa 3: Limpieza de Datos
- Tratamiento de valores únicos
- Tratamiento de valores faltantes
- Tratamiento de valores atípicos
- Justificación de decisiones

### Etapa 4: Vista Minable
- Normalización
- Discretización
- Numerización

### Etapa 5: Infografía
- Creación de infografía con hallazgos principales
